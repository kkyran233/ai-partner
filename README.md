# AI智能伴侣

这是一个基于 Streamlit 和 DeepSeek API 构建的 AI 智能伴侣应用。用户可以通过与 AI 进行对话，体验个性化的情感陪伴。

## 功能特性

- **个性化对话**：根据用户设定的伴侣昵称和性格特征，AI 将以符合设定的角色身份进行互动。
- **会话管理**：支持创建、保存、加载和删除多个会话记录。
- **实时交互**：采用流式输出方式实现实时对话效果。
- **界面友好**：使用 Streamlit 构建直观易用的 Web 界面。

## 技术栈

- Python 3.10
- [Streamlit](https://streamlit.io/)：用于快速搭建 Web 应用界面
- [OpenAI SDK](https://github.com/openai/openai-python)：调用 DeepSeek 大模型 API 实现对话功能
- JSON 文件存储：本地保存会话数据

## 安装与运行

访问浏览器中的地址即可开始使用。

## 使用说明

1. **设置伴侣信息**  
   在左侧边栏中输入伴侣的昵称和性格描述，点击输入框外任意区域自动保存。

2. **发起对话**  
   在底部输入框中输入你想说的话，按回车发送消息。

3. **管理会话**  
   - 新建会话：点击“新建会话”按钮可清空当前对话并开启新话题。
   - 查看历史：在“会话历史”列表中选择已有会话加载查看。
   - 删除会话：点击对应会话右侧的 ❌ 图标删除不需要的历史记录。
